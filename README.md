# Final-Project-University
My final project for my BSc at the University Of Leicester.
A Crowd-Counting project using the MCNN model (https://ieeexplore.ieee.org/document/7780439) adapted with the CBAM attention mechanism Using the ShanghaiTech dataset 

# Setup Guide

## Prerequisites
- Python 3.10 or higher
- Git

---

## 1. Clone the repository

```bash
git clone https://github.com/Hazza-02/Final-Project-University.git
cd Final-Project-University
```

---

## 2. Create and activate a virtual environment

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**macOS/Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 3. Install dependencies

```bash
pip install -r requirements.txt
```

> **GPU support (recommended for training):** If you have an NVIDIA GPU, install PyTorch with CUDA instead.
> Visit https://pytorch.org/get-started/locally/ and select your CUDA version, then run the command shown.
> CPU-only training on 400 epochs is very slow.

---

## 4. Download the ShanghaiTech Dataset

1. Download from (https://www.kaggle.com/datasets/tthien/shanghaitech)
2. Extract and place it so the structure looks like this:

```
Final-Project-University/
├── ShanghaiTech/
│   └── part_A/
│       ├── train_data/
│       │   ├── images/
│       │   ├── ground-truth/
│       │   └── density_maps/      ← generated by preprocess.py
│       └── test_data/
│           ├── images/
│           ├── ground-truth/
│           └── density_maps/      ← generated by preprocess.py
```

---

## 5. Preprocess — Generate Density Maps

Before training, you must generate density maps from the ground truth `.mat` files:

```bash
python preprocess.py
```

> Edit the paths in `preprocess.py` to point to your local ShanghaiTech directory if needed.

---

## 6. Train the model

```bash
python train.py
```

- Trains the **MCNN-CBAM** model by default (edit `train.py` to swap in `MCNN` for the baseline)
- Best model weights are saved to `Model Checkpoints/best_model_CBAM.pth`
- Training curves are plotted at the end of training

Key hyperparameters (editable in `train.py`):

| Parameter     | Value  |
|---------------|--------|
| Epochs        | 400    |
| Batch size    | 1      |
| Learning rate | 1e-6   |
| Optimiser     | SGD (momentum=0.9) |

---

## 7. Evaluate the model

```bash
python test.py
```

- Loads the saved model weights and runs evaluation on the test set
- Reports average **MSE Loss** and **MAE**
- Edit `model_path` in `test.py` to point to your saved `.pth` file
